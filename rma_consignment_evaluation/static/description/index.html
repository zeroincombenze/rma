Return Merchandise Authorization (RMA) from Consignent/Evaluation

Reso Merce Autorizzato (RMA) da C/Deposito o C/Visione

This module allows you to create a RMA from sale order, both for return both for sale.

RMA for sale moves product to Customer location and set quantity to invoice on sale order lines.

RMA for return moves product to internal stock. Notice this behavior is different from standard RMA which move products into specific RMA location.

Questo modulo permette di creare RMA sia per rientro merce che per vendita merce.

La richiesta di RMA per vendita sposta i prodotti nell'ubicazione clienti e attiva le quantità da fatturare nelle righe ordine.

Le RMA di reso spostano i prodotti nell'ubica interna si Stock. Notare che questo comportamenot differisce dal normale funzionamento delle RMA che spostano i prodotti in una specifica ubica interna.

Configuration | Configurazione

To configure RMA you need to:

Sales » Configuration » Sales Orders Types » Create a new sale order type with all the settings you want

The field Picking Auto Confirmation requires module sale_order_lot_selection

Per configurare il modulo:

Vendite » Configurazione » Tipi Ordine di vendita » Impostare la configurazione desiderata dell'ordine di vendita.

Il campo Auto conferma prelievo richiede il modulo sale_order_lot_selection.

Usage | Utilizzo

To use this module, you need to:

  1. Sales » Sales Orders » Orders
  2. Create a new sales order or select an existing one
  3. When the sales order is in 'Sales Order' state you can see in the status bar a button labeled  Create RMA , click it and a wizard will appear
  4. Modify the data at your convenience and click on  Accept  button
  5. As many RMAs as lines with quantity greater than zero will be created; Those RMAs will be linked to the sales order

Per creare l'ordine di rientro:

  1. Vendite » Ordini » Ordini
  2. Selezionare un ordine di c/deposito o c/visione
  3. Se l'ordine è confermato appare il bottone  Creare RMA ; fare click e seguire le istruzioni
  4. Modificare i dati in modo opportuno e fare click su  Accettazione 
  5. Viene creato l'ordine di rientro
